-----------------------------------------------
INTRODUCCIÓN
-----------------------------------------------

Este documento forma parte de un sistema simple
para gestionar notas académicas. El objetivo es
practicar el uso de Git y sus funcionalidades
básicas de control de versiones.

1. PROGRAMACIÓN

   - Estado: En progreso

2. BASES DE DATOS
   - Tema: Normalización
   - Conceptos: 1FN, 2FN, 3FN
   - Estado: Completado

3. DESARROLLO WEB
   - Tema: HTML y CSS
   - Conceptos: Estructura, estilos, responsive
   - Estado: En progreso
